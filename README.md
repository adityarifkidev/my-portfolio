## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)

1. **Clone the repository**
   ```bash
   git clone https://github.com/adityarifki/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```


1. **Update Personal Information**
